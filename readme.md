# const cli [personal]

generate a new project quickly by template

## how to use

``` sh
const init PROJECJ_NAME [type]
```

type:

- `h5`: html5
- `vue2`: vue2 `waiting`
- `nest`: nest.js `waiting`
- `mp-wx`: 微信小程序
- `addon`: Chrome插件 `waiting`
- `laravel`: Laravel
- ...and more will be supported!

## dev

``` sh
npm link
```
